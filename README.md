# 🌍 Travel & Tourism Website

A fully responsive and professional Travel & Tourism Website built using HTML, CSS, and JavaScript.

---

## 📌 Project Overview

This website allows users to:

- Explore popular travel destinations
- View tour packages
- Calculate package cost
- Book tours with form validation
- Experience responsive modern UI design

This project is frontend-only and built for learning and practice purposes.

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Live Server (for local development)

## 🚀 How To Run The Project

### Method 1: Using Live Server

1. Open project folder in VS Code
2. Right click on `index.html`
3. Click **Open with Live Server**
4. Website will open in browser

### Method 2: Directly Open

Double-click `index.html` file to open in browser.
